## Tentang Dummy Laravel

Dummylara ini dibuat untuk keperluan dummy pembuatann aplikasi. Jadi para developer laravel ketika akan membuat sebuah project, nantinya tidak perlu lagi membuat project dari awal. Karena nantinya dummylara ini akan dilengkapi dengan :

- User Management (CRUD), disini kita bisa melakukan management user
- Create Authorization and authentication, disini kita bisa melakukan setup autentikasi dan autorization kepada user.

## Catatan :

Aplikasi ini awalnya dibuat dengan tujuan agar saya secara pribadi bisa memiliki dummy ketika ingin membuat sebuah project. Namun juga bagi siapapun yang ingin menggunakannya, saya persilahkan tanpa ada batasan apapun.

Aplikasi ini jauh dari sempurna, maka bagi yang ingin menyempurnakan, maka saya sangat berterima kasih bagi siapapun yang akan berkontribusi
